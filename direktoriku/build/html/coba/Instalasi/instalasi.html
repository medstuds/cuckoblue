Instalasi¶

(1) Instalansi Librari Python pada Ubuntu/Debian¶

  1. Lakukan instalasi berikut ini:
$ sudo apt-get install python python-pip python-dev libffi-dev libssl-dev

$ sudo apt-get install python-virtualenv python-setuptools

$ sudo apt-get install libjpeg-dev zlib1g-dev swig

(2) Pembuatan Web Interface Cuckoo¶

  1. Lakukan instalasi mongo-db dengan menggunakan perintah:
$ sudo apt-get insttall mongo-db
  1. Instalasi postgresql
$ sudo apt-get insttall postgresql libpq-dev
  1. Instalasi apparmor tcpdump
$ sudo apt-get install tcpdump apparmor-utils

Tcpdump membutuhkan hak akses root, tetapi Cuckoo seharusnya tidak dijalankan sebagai root, lakukan pengaturan dengan:

$ sudo groupadd pcap
$ sudo usermod -a -G pcap cuckoo
$ sudo chgrp pcap /usr/sbin/tcpdump
$ sudo setcap cap_net_raw,cap_net_admin=eip /usr/sbin/tcpdump
  1. Pengunduhan ISO Windows 7 dan proses mounting ISO, unduh ISO, buat folder dan mounting:
$ wget https://cuckoo.sh/win7ultimate.iso
$ mkdir /mnt/win7
$ sudo mount -o ro,loop win7ultimate.iso /mnt/win7

(3) Instalasi dan Pengaturan Virtualbox¶

  1. Penambahan kunci repositori:
$ wget -q https://www.virtualbox.org/download/oracle_vbox_2016.asc -O- | sudo apt-key add -
$ wget -q https://www.virtualbox.org/download/oracle_vbox.asc -O- | sudo apt-key add -
  1. Penambahan repositori VirtualBox:
$ sudo add-apt-repository "deb [arch=amd64] http://download.virtualbox.org/virtualbox/debian $(lsb_release -cs) contrib"
  1. Instalasi VirtualBox 5.2:
$ sudo apt-get install virtualbox-5.2
  1. Instalasi M2Crypto:
$ sudo pip install m2crypto

(4) Instalasi dan Pengaturan Cuckoo-Virtualvenv.sh¶

  1. Install virtualenv:
$ sudo apt-get update && sudo apt-get -y install virtualenv
  1. Install virtualenvwrapper:
$ sudo apt-get -y install virtualenvwrapper
  1. Install pip for python3:
$ sudo apt-get -y install python3-pip
  1. pip3 install –user virtualenvwrapper:
$ echo "export VIRTUALENVWRAPPER_PYTHON=/usr/bin/python3" >> ~/.bashrc
$ echo "source ~/.local/bin/virtualenvwrapper.sh" >> ~/.bashrc
$ export WORKON_HOME=~/.virtualenvs
$ echo "export WORKON_HOME=~/.virtualenvs" >> ~/.bashrc
$ echo "export PIP_VIRTUALENV_BASE=~/.virtualenvs" >> ~/.bashrc
$ source ~/.bashrc

(5) Instalasi Cuckoo dan VM Cloack¶

  1. Masuk direktori cuckoo:
$ cd ..
$ cd cuckoo
$ virtualenv -p python2.7 cuckoo-test
$ . cuckoo-test/bin/activate
  1. Install VM Cloack:
$ pip install -U cuckoo vmcloak
  1. Instal pip setup tools:
$ pip install -U pip setuptools
  1. Instalasi kebutuhan VM Cloack dan Cuckoo:
$ sudo apt-get -y install build-essential libssl-dev libffi-dev python-dev genisoimage

$ sudo apt-get -y install zlib1g-dev libjpeg-dev

$ sudo apt-get -y install python-pip python-virtualenv python-setuptools swig

(6) Automatic VM Creation¶

  1. Pertama, mulailah dengan mendefinisikan dan membuat instance adapter jaringan Host-Only VirtualBox untuk digunakan VM:
$ vmcloak-vboxnet0
  1. Cek interface dengan menggunakan perintah:
$ ifconfig
  1. Kemudian membuat VM dan menginstal Windows secara otomatis, Cuckoo VM setidaknya memiliki 2GB memori dan dua atau lebih core CPU:
$ vmcloak init --verbose --win7x64 win7x64base --cpus 2 --ramsize 2048
  1. Membuat kloning dan snapshot dari VM:
$ vmcloak clone win7x64base win7x64cuckoo
  1. Lakukan instalasi vmcloack dengan Internet Explorer 11:
$ vmcloak install win7x64cuckoo ie11
  1. Instalasi snapshot untuk VM dengan ip yang digunakan:
$ vmcloak snapshot --count 4 win7x64cuckoo 192.168.56.101

(8) Pengaturan Jaringan dan Konfigurasi Routing¶

  1. Sebagai root, terapkan pada interface vboxnet0 dan interface keluar, penggunaan enss33 sebagai nilai dummy:
$ sudo sysctl -w net.ipv4.conf.vboxnet0.forwarding=1
$ sudo sysctl -w net.ipv4.conf.ens33.forwarding=1
  1. Konfigurasi pada file routing.conf, dengan mengubah interface yang digunakan oleh filter yaitu ens33:
$ sudo nano routing.conf
  1. Lakukan konfigurasi file reporting.conf pada bagian “ [mongodb]” pada baris “enabled = yes”:
$ sudo nano reporting.conf
  1. Menciptakan proses cuckoo router dengan tidak menjalankannya menggunakan perintah sudo:
$ cuckoo rooter --sudo --group pssn
  1. Menjalankan cuckoo web dengan server:
$ cuckoo web --host 0.0.0.0 --port 8080

(9) Menjalankan Cuckoo¶

  1. Setelah proses sebelumnya berhasil dilakukan, kemudian uji coba cuckoo dengan perintah:
$ cuckoo

2. Pada saat menjalankan cuckoo akan membutuhkan 3 tab terminal yang akan dijalankan secara bersamaan. Pada terminal pertama gunakan perintah berikut ini:

$ cuckoo rooter --sudo --group pssn
  1. Pada terminal kedua akan dijalankan dengan perintah dengan tujuan unutk mengakses ke media web server:
$ virtualenv -p python2.7 cuckoo-test
$ . cuckoo-test/bin/activate

4. Cara akses ke media web browser dengan membuka terminal baru (terminal ketiga) untuk melakukan ssh ke server. Jalankan perintah berikut terlebih dahulu pada tab ketiga:

$ cuckoo web --host 0.0.0.0 --port 8080
$ cuckoo
