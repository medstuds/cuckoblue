(6) Automatic VM CreationÂ¶

  1. Pertama, mulailah dengan mendefinisikan dan membuat instance adapter jaringan Host-Only VirtualBox untuk digunakan VM:
$ vmcloak-vboxnet0
  1. Cek interface dengan menggunakan perintah:
$ ifconfig
  1. Kemudian membuat VM dan menginstal Windows secara otomatis, Cuckoo VM setidaknya memiliki 2GB memori dan dua atau lebih core CPU:
$ vmcloak init --verbose --win7x64 win7x64base --cpus 2 --ramsize 2048
  1. Membuat kloning dan snapshot dari VM:
$ vmcloak clone win7x64base win7x64cuckoo
  1. Lakukan instalasi vmcloack dengan Internet Explorer 11:
$ vmcloak install win7x64cuckoo ie11
  1. Instalasi snapshot untuk VM dengan ip yang digunakan:
$ vmcloak snapshot --count 4 win7x64cuckoo 192.168.56.101
